Module Super_master_mind.Kheap

A Kheap.t is a mutable containers that retains the k smallest elements added to it overtime (and discard the others).

type 'a t
val sexp_of_t : ('a -> Sexplib0.Sexp.t) -> 'a t -> Sexplib0.Sexp.t
val create : k:Base.int -> compare:('a -> 'a -> Base.int) -> 'a t

create ~k ~compare creates a new kheap of size k, which will use the provided comparison function to identify the smallest elements. Once created, the size of the heap (k) and the comparison function (compare) may not be changed.

val add : 'a t -> 'a -> Base.unit

add t a will take a slot in t if a is smallest that the values already present in the kheap. If the resulting number of elements exceeds k, the bigger elements will be discarded. If a is bigger than the k elements already present in t, add t a will have no effect on t.

val to_list : 'a t -> 'a Base.list

Returns the elements of t ordered increasingly.
