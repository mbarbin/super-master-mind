Module Super_master_mind.Color_permutation

In order for the program to vary its opening guesses, we can choose to map the opening book through a color permutation, that is a bijection from the set of colors onto itself.

val cardinality : int

The number of different color permutations that exists.

Color permutations are ordered and indexed. The index may serve as efficient encoding.

val to_index : t -> int

to_index t returns the index of t in the color permutation set. It is guaranteed that 0 <= index < cardinality.

val of_index_exn : int -> t

of_index index returns the color permutation at the given index. Indices are expected to verify 0 <= index < cardinality. An invalid index will cause the function to raise.
