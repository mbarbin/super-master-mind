Module Super_master_mind.Codes

t is a container for several unordered Code.t. It is used to represent the set of possible solutions as the game goes on, shrinking from its original value (all possible combinations) down to the final guessed solution.

type t
val sexp_of_t : t -> Sexplib0.Sexp.t
val all : t

A t representing the set of all possible codes. Its size is equal to ~32K elements.

val size : t -> Base.int

Returns the number of codes contained in t.

val bits : t -> Base.Float.t

Returns the size of t expressed in a number of information bits.

val is_empty : t -> Base.bool

is_empty t = true is equivalent to size t = 0 but more efficient.

val filter : t -> candidate:Code.t -> cue:Cue.t -> t

Return a new set of codes in which only those code which evaluates to the given cue with the given candidate are kept. Note this may return an empty set if the original t contains no such code.

val to_list : t -> Code.t Base.list

Access the elements of t in the form of a list. The order of codes is unspecified.

val iter : t -> f:(Code.t -> Base.unit) -> Base.unit

Iter through the code contained in t.
