Module Super_master_mind.Code

A code is a sequence of colors, allowing repetition. A given solution to a game is a code, and each candidate submitted during the game is a code as well.

val create_exn : Hum.t -> t

Returns the efficient encoding of a given code. Raises if the size of the code does not match size.

val to_hum : t -> Hum.t

Returns the human readable representation of the code.

val size : int

The number of slots in the code. In this version of the game, this is 5.

val cardinality : int

The number of different codes that can be formed using all available colors, allowing repetition of colors.

Codes are ordered and indexed. The index may serve as efficient encoding.

val to_index : t -> int

to_index t returns the index of t in the total code set. It is guaranteed that 0 <= index < cardinality.

val of_index_exn : int -> t

of_index index returns the code at the given index. Indices are expected to verify 0 <= index < cardinality. An invalid index will cause the function to raise.
