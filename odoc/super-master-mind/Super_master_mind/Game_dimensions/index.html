Module Super_master_mind.Game_dimensions

So that the code is easier to test with smaller versions of the game, we make the rest of the code depend on global dimensions that are made available in this module. To change the default dimensions, the values must be set before evaluating any Lazy.t value that depends on it (which makes the use of this module brittle). If things get broken this will be easily spotted by the tests.
