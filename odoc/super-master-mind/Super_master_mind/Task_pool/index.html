Module Super_master_mind.Task_pool

To perform computation in parallel, we use Domainslib. This module is a thin wrapper on top of Domainslib.Task, which allows to make it easier to use it correctly, and allows for changing parameters from the command line, via the Config module.

type t
module Config : sig ... end
val with_t : Config.t -> f:(task_pool:t -> 'a) -> 'a

This wrapper takes care of setting up the task pool, and tearing it down when finished. It is meant to wrap the entire part of a command that will make use of the task pool.

val run : t -> f:(pool:Domainslib.Task.pool -> 'a) -> 'a

To be called each time you want to make use of the task pool. This wraps f within a call to Domainslib.Task.run.
